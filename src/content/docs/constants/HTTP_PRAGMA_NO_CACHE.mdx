Allows enabling/disbling of the &quot;Pragma: no-cache&quot; header.

Usage: [HTTP_PRAGMA_NO_CACHE, integer SendHeader]. When SendHeader is TRUE, the &quot;Pragma: no-cache&quot; header is sent by the script. This matches the default behavior. When SendHeader is FALSE, no &quot;Pragma&quot; header is sent by the script.
